# Auth Gateway Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Spring Boot auth gateway that supports Google login and issues stable internal OIDC identity for downstream platforms.

**Architecture:** The gateway is a Spring Boot application using Spring Authorization Server and Spring Security OAuth2 Client. It stores internal users in JPA, signs users in through Google, serves a portal page, and exposes OIDC metadata, token, JWKS, userinfo, and `/api/me` endpoints.

**Tech Stack:** Java 21, Spring Boot, Spring Authorization Server, Spring Security OAuth2 Client, Spring MVC, Thymeleaf, Spring Data JPA, H2 local database, PostgreSQL production driver, Maven Wrapper.

---

## File Structure

- `pom.xml`: Maven project generated by Spring Initializr.
- `src/main/java/dev/liangwen/authgateway/AuthGatewayApplication.java`: Boot entrypoint.
- `src/main/java/dev/liangwen/authgateway/config/SecurityConfig.java`: Authorization Server, OAuth2 login, OIDC, app security, token customization, registered clients, JWK source.
- `src/main/java/dev/liangwen/authgateway/config/IdentityProperties.java`: typed properties for portal apps and OIDC clients.
- `src/main/java/dev/liangwen/authgateway/user/AppUser.java`: internal gateway user entity.
- `src/main/java/dev/liangwen/authgateway/user/ExternalAccount.java`: provider account link entity.
- `src/main/java/dev/liangwen/authgateway/user/AppUserRepository.java`: user repository.
- `src/main/java/dev/liangwen/authgateway/user/ExternalAccountRepository.java`: provider account repository.
- `src/main/java/dev/liangwen/authgateway/user/UserIdentityService.java`: creates and updates users from Google claims.
- `src/main/java/dev/liangwen/authgateway/user/GatewayOidcUser.java`: authenticated principal whose name is the internal user id.
- `src/main/java/dev/liangwen/authgateway/web/MeController.java`: `/api/me`.
- `src/main/java/dev/liangwen/authgateway/web/PortalController.java`: portal page.
- `src/main/java/dev/liangwen/authgateway/web/AppRegistration.java`: configured app DTO.
- `src/main/resources/application.yml`: local defaults and env backed secrets.
- `src/main/resources/templates/portal.html`: navigation portal.
- `src/main/resources/static/styles.css`: portal styling.
- `src/test/java/dev/liangwen/authgateway/user/UserIdentityServiceTest.java`: user upsert behavior.
- `src/test/java/dev/liangwen/authgateway/config/IdentityPropertiesTest.java`: app/client property binding.
- `src/test/java/dev/liangwen/authgateway/web/MeControllerTest.java`: `/api/me` behavior.
- `README.md`: local run, Google OAuth setup, downstream integration notes.

        - ${JOB_CRM_REDIRECT_URI:http://localhost:3000/login/oauth2/code/auth-gateway}

        - ${JOB_CRM_LOGOUT_REDIRECT_URI:http://localhost:3000/job/}

      client-secret: ${INTERVIEW_CLIENT_SECRET:dev-interview-secret}

        - ${INTERVIEW_LOGOUT_REDIRECT_URI:http://localhost:3000/}

  endpoints:
    web:
      exposure:
        include: health,info
